Git fue creado en 2005 por Linus Torvalds, el mismo creador del sistema operativo Linux.

El proyecto surgió de la necesidad de un nuevo sistema de control de versiones (VCS) distribuido para gestionar el desarrollo del kernel de Linux. Antes de Git, el proyecto Linux utilizaba un sistema propietario llamado BitKeeper. Cuando la relación con BitKeeper se rompió, Torvalds se vio obligado a crear su propia herramienta en un tiempo récord, poniendo énfasis en la velocidad, la integridad de los datos y la capacidad de manejar proyectos masivos con muchos desarrolladores colaborando de forma descentralizada.

Hoy en día, Git es el Sistema de Control de Versiones Distribuido (DVCS) más popular del mundo, siendo la base de plataformas de desarrollo colaborativo como GitHub, GitLab y Bitbucket.
